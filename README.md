# Dragon Ball DB
Questo progetto tratta di un database di dragon ball fatto in excel.

## Cosa Serve Per Funzionare
- Excel / Fogli / Openoffice
- Qualsiasi sistema operativo

## Come Funziona
Illustro di seuito con le loro tabelle con i loro campi.
- Techniques: 
  - id
  - name
  - canon
  - type
  - damage
  - radius
  - ep
  

- Chtech: 
  - id
  - character
  - technique


- Characters:
  - keys 
  - name
  - hp
  - ep
  - atk
  - def
  - recharge
  - god
